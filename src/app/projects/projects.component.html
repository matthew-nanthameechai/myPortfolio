Recruiter

react - javascript - python

Recruiter is a prototype we developed at Provoke Solutions 2 day Hackathon where we used generative AI to show how it can be used to automate the recruitment screening process.The prototype we developed uses generative AI that reads the job description, the company info and the applicants CV to create tailored questions. This could be used to call the applicant and have AI ask the applicant the questions it created and transcribe the replies to text.
